## Getting Started

### Prerequisites

* Python 3.8 with virtualenv

### Installation

* create virtualenv named `venv`

```sh
virtualenv venv
```

* activate environment and install requirements.txt

```sh
source venv/bin/activate
pip install -r requirements.txt
```

* Run server

```sh
python manage.py runserver
```
